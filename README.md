# React + TypeScript + Vite

The Project sees a Command Palette component task. The screen lands on a page where on clicking Cmd + k or Ctrl + k opens a search box. Here, user can search the commands and sees teh label and keywords of that command. The up-down arrow navigates through the list and on clicking enter the respective action function is called. On clicking the Escape key the command palette closes.

## Steps to reproduce:

1. Unzip the file and open the folder
2. Run the command: "npm i"
3. Followed by command: "npm run dev"
